# Reference Index

The reference section is the shared vocabulary and notation layer for the
documentation. It should remain asset-class neutral unless a term or formula is
inherently asset-class specific.

## Canonical Files

1. `docs/reference/LEXIS.md`  
   Platform vocabulary, market-data terms, risk-factor language, and
   asset-class terms.

2. `docs/reference/FORMULAS.md`  
   Common valuation, curve, credit, FX, equity, commodity, option, risk, and
   LSMC formulas.

3. `docs/reference/SOURCES.md`  
   Public references and source classes used by the documentation.

## Maintenance Rule

Do not create separate glossary, formula-sheet, or source files inside
asset-class folders. When a concept is reusable, add it here and link to it from
the relevant chapter.

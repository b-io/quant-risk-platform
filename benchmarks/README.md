### Benchmarks Directory

This directory contains C++ source files for benchmarking the performance of the Quant Risk Platform.

#### Key Files

- `benchmark_portfolio.cpp`: A benchmarking tool used to measure the performance of portfolio valuation and risk calculations across different scales.

#### Usage

Benchmarks are typically compiled and run as part of the C++ build process. To run benchmarks, ensure the project is built and then execute the benchmark binaries found in the build directory.
